# ESP32 Rain Station

## Giới thiệu
Dự án trạm mưa sử dụng ESP32.
Thiết bị phát hiện mưa và gửi thông báo về điện thoại.

## Phần cứng sử dụng
- ESP32 Dev Module
- Cảm biến mưa
- Nguồn 5V (có sạc pin)
- Dây nối

## Chức năng
- Phát hiện mưa
- Gửi cảnh báo về điện thoại
- Hoạt động độc lập

## Cách sử dụng
1. Cấp nguồn cho ESP32
2. ESP32 kết nối WiFi
3. Khi có mưa → gửi thông báo

## File trong project
- esp32_rain_station.ino : Code chính
- README.md : Mô tả dự án

## Ghi chú
Dự án phục vụ học tập và nghiên cứu ESP32.
